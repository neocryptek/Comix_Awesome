Comix [Pure Awesome!] is an addon for World of Warcraft (WoW).

Adds /awesome version of "Comix's":http://wow.curse.com/downloads/wow-addons/details/comix-the-return.aspx /fail.

Usage: /awesome <channel> <text>

Where <channel> is a chat channel (such as "guild" or a player's name) and <text> is the message you want to send to <channel>, where all instances of "~~" are replaced with "[Pure Awesome]":http://www.wowhead.com/?spell=58783.

Note that "Comix":http://wow.curse.com/downloads/wow-addons/details/comix-the-return.aspx is a required dependency for this addon.
